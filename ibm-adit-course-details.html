Course Details

Directorate General of Training (DGT) under Ministry of Skill Development & Entrepreneurship offers a range of Vocational Training courses catering to the need of different sectors of economy/ Labour market.

IT, Networking and Cloud course is very much essential in the current scenario due to a lot of demand in software industries where every industry is using cloud as base. The course is for two years duration.

To apply for this course candidates have to satisfy any one of eligibility criteria.

  1. X/XII Std pass with NTC (NCVT certificate in relevant trade) of any one year or Two-year Trade with minimum 60% marks/First Class.
  2. Candidates pursuing CTS, in NSTI / ITI who have appeared or appearing in their final exams can also apply, provided they should have scored a minimum of 60% marks in their CTS trade.
  3. XII Std pass with minimum 60% marks/First Class.
  4. X std pass with three-year diploma from recognized board and minimum 60% marks/First class in Diploma.
  5. Any regular degree from recognized university with minimum 60% marks/First class

Duration and Syllabus of the course

In the first year there are five core modules each module is credit base and employable.

Each module is of 320 hours and is very much independent on the other hand in second year the trainee will be taking two elective modules out of three electives each of 320 hours and will be doing on the job training in Industry for 800 hours.

In addition the trainees will pick up employability skills for 160 hours.

  • FIRST YEAR

    Syllabus For First Year

    In first year there are five core modules each of 320 hours.

    • Core Module 1-Computer Hardware Maintenance
      In first module the trainee will learn about computer hardware and how to maintain it.
    • Core Module 2-Computer Networking
      This module deals with some basic and advance level of computer networking , Data Communications System and its components and so on.
    • Core Module 3-Web Designing
      In this module the trainee will gain some knowledge about web designing and will able to create any static and dynamic website.
    • Core Module 4-Web Development
      This module is related to the previous module but on advance level by the help of this module trainee will get exposed to programming concepts of JAVA , Create multimedia applications and many more.
    • Core Module 5-Business Data Analytics
      In this module the trainee will get some knowledge about business analytics, Develop business intelligence and related stuffs.

    Assessment

    • The Internal assessment during the period of training will be done by Formative assessment method by testing for assessment criteria listed against learning outcomes.
  • SECOND YEAR

    Syllabus For Second Year

    In the second year there are three elective modules each of 320 hours and trainee have to choose any two of them.

    After completition of these modules trainee will be go for the job training in same year and also paid for this.

    • Elective Module 1-Cloud Application Developer
      In this module the trainee will learn to Use of Cloud platform and its service , Application development on the cloud (using Eclipse, Node.js) and so on.
    • Elective Module 2-Cloud Enterprise Developer
      In this module the trainee will learn to Build a web application with the Express framework , Hands-on on Cloud Integration and many more.
    • Elective Module 3-Web Development using Java
      This module deals with the better understanding of object-oriented approach in programming , Develops the Web Pages using java and deploying the pages in Web server and related stuffs.

    Assessment

    • The Internal assessment during the period of training will be done by Formative assessment method by testing for assessment criteria listed against learning outcomes.
    • The final assessment will be in the form of summative assessment method. The All India Trade Test for awarding Technical Diploma will be conducted by NCVT at the end of each Module/year as per guideline of Govt of India. The learning outcome and assessment criteria will be basis for setting question papers for final assessment.
